Thero

Participated in the creation of Thero's resource directory. Created user sign up roles, which were dependent on user selection. Each role displays a different navigation dependent on the role. Also dealt with sorting CSF fiel fields on upload through Rails Admin. Practitioner profile cards on the map were generated using Polymer.

  • Client: Thero
  • Date: December 2015
  • Languages: Ruby on Rails, Polymer

ShelterFriends

ShelterFriends was a group project between myself and 4 other LEARN Academy students. This was for our final class project. Our aim was to create an interface that would easily allow users access to see pets avaiable for adoption. And also to create an easy to use interface for shelters to maintain their available pets.

  • Website: ShelterFriends Repository
  • Date: November 2015
  • Languages: Ruby on Rails

Tic Tac Toe

One of the very first programs that I wrote. A very simple game of Tic Tac Toe written in JavaScript, HTML and CSS. Players input their names before starting the game. The starting player is then randomly chosen. Players play until someone wins or ends in a draw.

  • Website: Tic Tac Toe Repository
  • Date: September 2015
  • Languages: JavaScript, HTML, CSS
